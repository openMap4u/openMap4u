openMap4u

  • visualizes data: in flexible ways.
  • supports: Gif, Png, Jpeg, and SVG format.

Examples

openMap4u is

  • easy to use: you don't have to be a rocket scientist.
  • embedable: works together with mainstream JEE technologies like JSF, EL, ... .
  • lightweight: core API has no dependencies (<100 classes, <84KB size).
  • extensable: Java (>7x) based (well documented) API.
