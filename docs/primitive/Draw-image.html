How to draw an image

Obtain an image primitive (with the configured default style). {% highlight java %} OpenMap4u om4u = new OpenMap4u(); ... SetAreaOfInterestOrDrawOrWriteable myCanvas = om4u.getCanvas(4, 1); ... Image myImage = om4u.get(Image.class); {% endhighlight %} Set the image to draw {% highlight java %} myImage.path("image.png"); {% endhighlight %} Set the position where to draw {% highlight java %} myImage.center(0.5,0.25); {% endhighlight %} And last but not least draw it {% highlight java%} myCanvas.draw(myImage); {% endhighlight%} Write the result: {% highlight java%} myCanvas.write("myImage.png"); {% endhighlight %} Which results in the following output.
But wait you can do it much faster and simpler with the fluent builder pattern {% highlight java%} myCanvas.draw( om4u.get(Image.class).path("myImage.png").center(0.5,0.25) ).write("myImage.png");; {% endhighlight %}
