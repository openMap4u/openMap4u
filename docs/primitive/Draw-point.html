How to draw a point

You cannot draw a point.

Why???

Well a point has no size So you cannot see it!!!

But you can draw
  • a line-,
  • a polygon-,
  • an image- and or
  • a text primitive
at a specific position as a point representation instead.

Text example: {% highlight java %} myText.center(0.5,0.7); {% endhighlight %} Image example: {% highlight java %} myImage.center(0.5,0.7); {% endhighlight %} Line example: {% highlight java %} myLine.center(0.5,0.7); {% endhighlight %} Polygon example: {% highlight java %} myPoylgon.center(0.5,0.7); {% endhighlight %}
