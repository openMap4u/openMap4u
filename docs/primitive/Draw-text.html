How to draw a text

Obtain a text primitive (with the configured default style): {% highlight java %} OpenMap4u om4u = new OpenMap4u(); ... SetAreaOfInterestOrDrawOrWriteable myCanvas = om4u.getCanvas(4, 1); ... Text myText = om4u.get(Text.class); {% endhighlight %} Set the text to draw: {% highlight java %} myText.text("Hello world"); {% endhighlight %} Set the position where to draw: {% highlight java %} myText.center(0.5,0.25); {% endhighlight %} Draw it: {% highlight java %} myCanvas.draw(myText); {% endhighlight %} Write the result {% highlight java%} myCanvas.write("myText.png"); {% endhighlight %} Which results in the following output.
But wait you can do it much faster and simpler with the fluent builder pattern {% highlight java %} myCanvas.draw( om4u.get(Text.class).text("Hello world").center(0.5,0.25) ).write("myText.png"); {% endhighlight %}
