How to draw a line

Obtain a canvas and a line primitive (with the configured default style). {% highlight java %} OpenMap4u om4u = new OpenMap4u(); ... SetAreaOfInterestOrDrawOrWriteable myCanvas = om4u.getCanvas(4, 1); ... Line myLine = om4u.get(Line.class); {% endhighlight %} Set the line from a start to an end point (xStart, yStart, xEnd, yEnd). {% highlight java%} myLine.line(.25, .125, 3.75, 0.75); {% endhighlight %} Draw the line {% highlight java%} myCanvas.draw(myLine); {% endhighlight %} Write the result {% highlight java%} canvas.write("myLine.png"); {% endhighlight %} Which results in the following output.
But wait you can do it much faster and simpler with the fluent builder pattern {% highlight java%} myCanvas.draw( om4u.get(Line.class).line(.25, .125, 3.75, 0.75) ).write("myLine.png"); {% endhighlight %}
