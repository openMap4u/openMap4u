How to use openMap4u

Create a new openMap4u instance (with the default settings). {% highlight java%} OpenMap4u oM4u = new OpenMap4u(); {% endhighlight %} Optional override the default settings.

Create a rectangular drawing canvas with the given width (=4 cm) and height (=2 cm). {% highlight java %} OverrideDrawOrWriteable myCanvas = oM4u.getCanvas(4.0, 1.0); {% endhighlight %} Optional set the area of interest.

Draw "Hello world" at the given position on the canvas. {% highlight java %} myCanvas.draw( oM4u.get(Text.class).text("Hello world").center(0.5,0.25) ); {% endhighlight %} Write the output. {% highlight java %} canvas.write("myText.png"); {% endhighlight %} Which results:
But wait you can do it much faster and simpler with the fluent builder pattern {% highlight java %} new OpenMap4u().canvas(4.0,1.0).draw( om4u.get(Text.class).text("Hello world").center(0.5,0.25) ).write("myText.png"); {% endhighlight %}
